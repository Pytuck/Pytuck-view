# CHANGELOG

## [0.3.0] - 2026-02-01

### 新增功能

#### 数据编辑功能
- 添加表数据的增删改功能（插入、更新、删除行）
- 添加表结构编辑功能（重命名表、修改表备注、修改列备注）
- 添加多行文本编辑器，支持长文本内容编辑
- 添加数据验证机制，优化必填字段提示
- 添加错误消息复制功能，便于问题排查

#### 国际化增强
- 实现前端国际化消息的键值映射机制
- 添加数据编辑功能的完整国际化支持
- 添加字段必填提示的国际化支持
- 添加复制功能的国际化支持
- 添加数据操作失败消息的国际化支持
- 添加前端国际化 key 字段，后端可选填

#### 后端改进
- 添加主键重复错误处理（`DuplicateKeyError`）
- 优化表行计数逻辑，使用 `storage.count_rows()` 方法
- 优化数据库操作错误日志和异常处理
- 添加表和行操作的请求体模型（Pydantic）

### 界面优化

- 优化表编辑功能界面布局
- 更新按钮样式以改进视觉效果和对齐方式
- 优化错误提示显示效果

### 修复

- 修改开发服务器启动端口为 54540

### 文档与其他

- 添加 LICENSE 文件
- 修复 PyPI 上传命令中的包名格式
- 添加国际化要求和翻译类组织规范到 CLAUDE.md

### 依赖更新

- 更新 pytuck 依赖版本
- 更新 setuptools 版本

---

## 变更摘要

本版本是一个重要的功能更新版本，主要新增了 **数据编辑功能**，用户现在可以：

1. **编辑表数据**：插入新行、更新现有行、删除行
2. **编辑表结构**：修改表名、表备注、列备注
3. **更好的错误处理**：主键冲突等错误现在有更友好的提示

同时，国际化系统得到了显著增强，为前端 UI 提供了完整的多语言支持。
